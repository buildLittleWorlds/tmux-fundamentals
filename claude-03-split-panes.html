Split Panes

Lesson 3 of 10 Beginner Track: tmux + Claude Code

Panes divide a single tmux window into multiple terminal areas, each running its own shell. This is where tmux becomes genuinely powerful for development—you can watch Claude Code work in one pane while editing, testing, or browsing files in another.

What Are Panes?

A pane is a rectangular section of your tmux window with its own independent shell. You can split a window horizontally (top/bottom) or vertically (left/right), and move between panes with the prefix key plus an arrow key.

tmux uses two split directions:

  • Ctrl+J " — splits horizontally (creates a top pane and a bottom pane)
  • Ctrl+J % — splits vertically (creates a left pane and a right pane)

Exercise

1 Confirm you’re in your tmux session

You should be inside the tutorial session from the previous lessons. Check for the status bar at the bottom.

2 Split horizontally (top/bottom)

Press Ctrl+J " (prefix, then the double-quote key).

Your window divides into two panes stacked vertically. The cursor lands in the bottom pane.

3 Move between panes

Jump to the top pane:

Ctrl+J ↑ (prefix, then the up arrow key)

The active pane border highlights so you can see which pane has focus.

4 Close a pane to reset

Close the current pane by exiting its shell:

exit

You’re back to a single pane.

5 Split vertically (left/right)

Press Ctrl+J % (prefix, then the percent key).

Your window divides into two side-by-side panes. The cursor lands in the right pane.

6 Navigate between the panes

Move left and right with arrow keys:

Ctrl+J ← to move left  /  Ctrl+J → to move right

7 Run commands in each pane

In the left pane, run:

echo "I'm the left pane"

Move to the right pane (Ctrl+J →) and run:

echo "I'm the right pane"

Each pane runs its own independent shell—commands in one don’t affect the other.

Checkpoint

  • You have two panes side by side, each with its own shell
  • You can move between them with Ctrl+J ← and Ctrl+J →
  • Commands run in one pane don’t affect the other

Quick Reference

Command Action
Ctrl+J " Split horizontally (top/bottom)
Ctrl+J % Split vertically (left/right)
Ctrl+J ↑↓←→ Move between panes
Ctrl+J x Close current pane

Why vertical splits matter for AI workflows: The left/right split (Ctrl+J %) is the foundation of the supervisor workflow you’ll learn in Lesson 5. You’ll run Claude Code on one side and keep a shell on the other for testing, file browsing, and git operations.
