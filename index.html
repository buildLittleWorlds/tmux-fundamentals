See everything. Control everything. Interactive tutorials for mastering tmux with Claude Code and Codex CLI.

Curriculum Overview

Five tracks of hands-on tutorials that teach you tmux by building real tools with AI coding assistants. Start with a beginner track to learn the fundamentals, explore the guardrails that make autonomous agents safe and fast, then level up with advanced multi-agent orchestration and developer automation. Each track has 10 lessons that follow a Concept → Exercise → Checkpoint pattern.

Getting Started

Before starting any track, run through the Setup & Configuration page to configure your tmux prefix key, terminal settings, and AI CLI tools. It takes about five minutes.

Prerequisites:

  • tmux 3.0+ installed (brew install tmux on macOS)
  • An AI CLI installed: Claude Code or Codex CLI (or both)
  • A terminal emulator
  • Basic command line familiarity

Beginner Tracks

Choose one to start. Both teach the same tmux fundamentals — sessions, panes, windows, copy mode — while you build a working td todo CLI with your chosen AI assistant.

Intermediate Track

After completing a beginner track, this track teaches the engineering mindset for working with autonomous AI agents. It connects the tmux skills you learned above with the guardrail strategies that make headless and background agents safe.

Guardrails for Speed

10 lessons · Intermediate

The higher you climb the autonomy ladder, the more guardrails you need. Learn to engineer permission tiers, hooks, type systems, test automation, token budgets, and architectural constraints into a reusable pipeline that lets AI agents work fast and safely.
