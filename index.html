Curriculum Overview

Four tracks of hands-on tutorials that teach you tmux by building real tools with AI coding assistants. Start with a beginner track to learn the fundamentals, then level up with advanced multi-agent orchestration and developer automation. Each track has 10 lessons that follow a Concept → Exercise → Checkpoint pattern.

Getting Started

Before starting any track, run through the Setup & Configuration page to configure your tmux prefix key, terminal settings, and AI CLI tools. It takes about five minutes.

Prerequisites:

  • tmux 3.0+ installed (brew install tmux on macOS)
  • An AI CLI installed: Claude Code or Codex CLI (or both)
  • A terminal emulator
  • Basic command line familiarity
