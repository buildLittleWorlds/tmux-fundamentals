Split Panes

Lesson 3 of 10 Beginner Track: tmux + Codex CLI

Panes split a single tmux window into multiple areas, each with its own shell. This is where the real power comes from—code in one pane, Codex in the other, both visible at the same time.

Horizontal and Vertical Splits

tmux supports two types of splits:

  • Horizontal split (Ctrl+J ") — divides the window into top and bottom panes
  • Vertical split (Ctrl+J %) — divides the window into left and right panes

You move between panes with the prefix key followed by an arrow key. Each pane runs its own independent shell, so you can have different commands running side by side.

What you’ll do: Try both split directions, navigate between panes with arrow keys, and end up with a left/right layout—the configuration you’ll use for the supervisor workflow in the next lesson.

Exercise

1 Make sure you’re inside tmux

You should be in your codex-lab session. If not:

tmux attach -t codex-lab
2 Split horizontally (top/bottom)

Press the prefix key followed by the double-quote key:

Ctrl+J "

Your window is now divided into a top pane and a bottom pane. The cursor starts in the bottom pane.

3 Move between panes

Navigate to the top pane using the prefix key and the up arrow:

Ctrl+J ↑

The cursor jumps to the top pane. Each pane has its own independent shell prompt.

4 Close the top pane to reset

Type exit in the top pane to close it and return to a single pane:

exit

You’re back to one full-size pane.

5 Split vertically (left/right)

Now split with the percent key:

Ctrl+J %

You now have two panes side by side. The cursor is in the right pane. This left/right layout is the one you’ll use most—your code on one side, Codex on the other.

6 Navigate left and right

Move between the two panes with arrow keys:

Ctrl+J ← # move left Ctrl+J → # move right

Try switching back and forth a few times to build the muscle memory. This is how you’ll move between your testing pane and the Codex pane throughout this track.

Checkpoint

  • You have two panes side by side (left and right)
  • Each pane has its own independent shell
  • You can move between them with Ctrl+J ← and Ctrl+J →

What’s Next: You’ve got two panes side by side—now it’s time to put something in them. In Lesson 4: Launch Codex CLI, you’ll start Codex in one pane and begin building your td CLI tool in the other.
