Copy Mode (Scrolling Back)

Lesson 8 of 10 Beginner Track: tmux + Claude Code

Terminal output scrolls away fast—especially when Claude is generating code. Copy mode freezes the pane and lets you scroll back through history, search for specific text, and copy content. This is essential for reviewing Claude’s longer responses and finding specific code blocks in its output.

What Is Copy Mode?

Normally, your terminal is in interactive mode—keystrokes go to the running program. Copy mode temporarily turns your pane into a read-only viewer where keystrokes navigate and search instead. Think of it like a built-in pager (similar to less) for everything that’s appeared in the pane.

When you enter copy mode, you’ll see a position indicator in the top-right corner of the pane showing where you are in the scrollback buffer. Your shell or running program is paused from your perspective—it keeps running, but new output won’t scroll the view until you exit copy mode.

Exercise

1 Generate some output to scroll through

In either pane, run this command to produce 100 lines of output:

for i in {1..100}; do echo "Line $i: The quick brown fox"; done

Most of this output has scrolled past the visible area. Normally it’s gone—but not with copy mode.

2 Enter copy mode

Press:

Ctrl+J [

Notice the indicator in the top-right corner showing your position. The pane is now frozen—you can navigate freely.

3 Navigate the scrollback

Try each of these navigation commands:

# Scroll line by line ↑ / ↓ # Scroll a full page Page Up / Page Down # Jump to the very top (Meta + Shift+,) M-< # Jump to the very bottom (Meta + Shift+.) M->

Meta key: M- means Meta. On macOS, press Option (if your terminal sends Option as Meta) or press Esc then the key. On Linux, use Alt. So M-< is Option+Shift+, on macOS, or Esc then Shift+,.

Scroll up until you find Line 1: The quick brown fox at the top of the output.

4 Search for text

Still in copy mode, search for specific text:

# Search forward (incremental) Ctrl+S then type: fox # Jump to next match n # Jump to previous match N # Search backward (incremental) Ctrl+R then type: brown

The search is incremental—matches highlight as you type. Press Enter to stop at the current match, then n or N to cycle through results. This is invaluable when Claude produces long output and you need to find a specific function, error message, or code block.

5 Exit copy mode

When you’re done browsing, exit copy mode:

q

You can also press Escape. The pane returns to interactive mode and any new output appears normally.

Checkpoint

  • Ctrl+J [ enters copy mode
  • You can scroll through the full output history
  • Ctrl+S and Ctrl+R search forward and backward
  • q exits copy mode

Copying and Pasting Text

Copy mode isn’t just for reading—you can select and copy text directly within tmux, then paste it into any pane:

Step Key Action
1 Space Start selection at the cursor position
2 Arrow keys Move to expand the selection
3 Enter Copy the selection (and exit copy mode)
4 Ctrl+J ] Paste the copied text into any pane

Why this matters with Claude: When Claude produces a code block you want to use elsewhere—say, a command it suggested or a config snippet—you can enter copy mode, select just that text, copy it, switch panes, and paste it. No mouse required.

Copy Mode Quick Reference

Command Action
Ctrl+J [ Enter copy mode
↑ / ↓ Scroll line by line
Page Up / Page Down Scroll by page
M-< / M-> Jump to top / bottom
Ctrl+S Search forward (incremental)
Ctrl+R Search backward (incremental)
n / N Next / previous match
q or Escape Exit copy mode
